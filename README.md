## Description
"GameClockChallenge" is a Python project that combines entertainment and utility into one versatile application. It features a customizable alarm clock integrated with engaging games. Users can set alarms to their desired times, with the unique twist that the alarm can only be turned off by successfully completing a designated game challenge. This innovative approach encourages users to wake up on time while stimulating their minds with interactive gameplay. With "GameClockChallenge," mornings become more enjoyable and productive, offering a seamless blend of fun and functionality.

## Features
- Set alarms with customizable times.
- Alarm can only be turned off by solving game challenges.
- Various game options including number guessing, rock-paper-scissors, series identification, and mindbreaker.
- Interactive user interface using Tkinter.
- Real-time clock display.
- Threading for smooth alarm functionality without freezing the GUI.

## How to Use
1. Clone the repository to your local machine.
2. Ensure you have Python installed.
3. Run the main script `GameClockChallenge.py`.
4. Set your alarm time and choose a game challenge.
5. Enjoy waking up to a fun challenge every morning!

## Files Included
- `GameClockChallenge.py`: Main Python script containing the application logic.
- `words.txt`: Text file containing words used for the mindbreaker game.
- `alpha.txt`: Text file containing alphabets for encryption in the mindbreaker game.

## Requirements
- Python 3.x
- Tkinter library
